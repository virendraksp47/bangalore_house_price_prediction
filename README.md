# 🏠 Bangalore House Price Prediction

Machine Learning project to predict Bangalore house prices using regression algorithms.

## 📌 Project Workflow

- Data Cleaning
- Missing Value Handling
- Feature Engineering
- Outlier Removal
- Model Building
  - Linear Regression
  - Ridge Regression
  - Lasso Regression
- Model Evaluation

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## 📊 Models Used

| Model | Purpose |
|---------|---------|
| Linear Regression | Baseline Model |
| Ridge Regression | L2 Regularization |
| Lasso Regression | L1 Regularization & Feature Selection |

## 📈 Goal

Predict house prices accurately based on location, size, sqft, and other property features.
